# Finance Alerts Channel Automation

This folder contains n8n workflows used to automate the Finance Alerts Telegram channel.

The automation posts:

- Finance news (Gold, Stocks, Crypto)
- Hourly price updates for:
  - BTC
  - ETH
  - SOL
  - BNB
  - Gold

Fully automated Telegram finance alert system built with n8n.
The system fetches financial news and live asset prices and publishes updates automatically.
